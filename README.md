# GiroFest 2026

Estrutura do site:

- `index.html`: pagina principal do site.
- `css/styles.css`: estilos visuais, paleta, responsividade e layout.
- `js/main.js`: envio do formulario e validacao com Google Apps Script.
- `assets/images`: logo e arte principal do GiroFest.
- `assets/videos`: videos recebidos do WhatsApp, guardados para uso futuro.
- `pages/camp_meet_2026-single-file.html`: copia antiga em arquivo unico, mantida como referencia.

Para abrir o site, use `index.html`.
